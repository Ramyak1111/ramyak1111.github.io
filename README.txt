Ramya portfolio — static bundle
------------------------------

Serve this folder locally (opening the HTML file directly can break some assets):

  cd %USERPROFILE%\Downloads\Ramya-Portfolio
  python -m http.server 8080

Then open:
  http://localhost:8080/                 (index.html)
  http://localhost:8080/ramya-portfolio.html

Included in this copy:
  images\ramya-hero.svg
  privacy.html

Optional files you can add later:
  resume.pdf

Notes:

  - `resume.pdf` has been updated from `Ramya Hr Resume.pdf`.
  - The site is positioned for HR opportunities in Canada and the USA.
